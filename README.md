# InAmigos Foundation Webpage

## Overview
This repository contains the source code for the official webpage of the **InAmigos Foundation**, a Central Government licensed Section 8 non-profit organization headquartered in Bilaspur, Chhattisgarh. 

Established on September 23, 2020, by Mr. Govind Shukla, the foundation has touched over 50,000 lives across 28 Indian states through a variety of initiatives focusing on education, food security, women empowerment, animal welfare, environmental conservation, and skill development.

## Flagship Initiatives
The InAmigos Foundation drives its mission through six interconnected projects:
- **BachpanShala:** Education for underprivileged children.
- **Seva:** Food security and welfare programs.
- **Udaan:** Women empowerment and self-reliance.
- **Jeev:** Animal welfare and protection.
- **Prakriti:** Environmental conservation and sustainability.
- **Vikas:** Skill development and youth enablement.

## Technical Details
This repository contains the static web assets for the foundation's landing page. 
- `index.html`: The main structure and content of the webpage.
- `style.css`: The styling rules ensuring a responsive and modern design.
- The project is deployed on Vercel and does not require a build step as it uses vanilla HTML/CSS.

## Running Locally
To preview the website locally, simply open the `index.html` file in any modern web browser, or use a local server like Live Server (VS Code extension) or Python's `http.server`.

```bash
# Using Python
python3 -m http.server 8000
```
Then navigate to `http://localhost:8000`.
